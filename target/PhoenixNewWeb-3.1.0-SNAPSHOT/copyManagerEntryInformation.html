Welcome to the Phoenix Copy Manager

It is a tool for Business Administrators in Phoenix which allows to move and copy data between folders.

Training material

To copy or move data you have to:
    1. Select and open affected folder from left tree panel
    2. In the middle area select by checkboxes folders and documents you would like to move or copy
    3. Expand right tree panel to see target folder
    4. Drag all selected items from the middle area and drop them into target folder in the right tree panel
    5. In the bottom panel solve all met conflicts and decide to move or copy items
